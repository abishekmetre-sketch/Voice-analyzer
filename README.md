# Voice Analyzer (MVP)

This is a **prototype** inspired by Phonalyze.

Features:
- Upload audio
- Compute pitch, jitter, shimmer, HNR, CPP
- Show waveform, pitch contour, spectrogram
- Export metrics as CSV
